# Investigation-portfolio

This repository showcases my SOC analyst investigation portfolio built from CyberDefenders SOC Analyst Tier 1–3 labs. Each case study walks through realistic blue-team scenarios, including alert triage, log and SIEM analysis, endpoint and network forensics, and incident response actions based on CyberDefenders’ real-world style simulations. The goal is to demonstrate job-ready skills in threat hunting, evidence-driven analysis, and clear reporting that align with modern SOC workflows and the Certified CyberDefender training path.
